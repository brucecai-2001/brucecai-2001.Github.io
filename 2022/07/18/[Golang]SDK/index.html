Golang---SDK

Author: Xinyu Date: July 18, 2022  22:09:22 Category: Go

对于 SDK（Software Development Kit，软件开发工具包），不同场景下有不同的解释。但是对于一个 Go 后端服务来说，SDK 通常是指封装了 Go 后端服务 API 接口的软件包，里面通常包含了跟软件相关的库、文档、使用示例、封装好的 API 接口和工具。调用 SDK 跟调用本地函数没有太大的区别，所以可以极大地提升开发者的开发效率和体验。
