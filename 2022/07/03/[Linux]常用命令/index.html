[Linux]常用命令

Author: Xinyu Date: July 3, 2022  22:51:11 Category: Linux

防火墙

查询防火墙状态: sudo ufw status
开启: sudo ufw enable
关闭: sudo ufw disable
打开端口: sudo ufw allow portNum

查看进程

ps -ef | grep xxx

TCP 的连接状态查看

netstat -napt

查看当前系统的路由表。

route -n
